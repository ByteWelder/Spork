# Contributing

## Java 7

Only use Java 7 features that are fully supported on Android:

Supported:
- Diamond operator
- String switch
- Multiple-catch
- Underscores in number literals
- Binary literals

NOT supported:
- `try-with-resources`
- `java.lang.SafeVarargs`

See http://stackoverflow.com/a/13550632/3848666 for more information.
